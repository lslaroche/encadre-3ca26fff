# encadré

Encadré est une appli web qui permet aux locataires parisiens de vérifier si leur loyer respecte l’encadrement des loyers. 
L'utilisateur entre son adresse, quelques infos (surface, meublé/non meublé, pièces, époque) et son loyer : l’outil compare avec les loyers de référence et affiche un verdict. Le résultat peut être partagé (paramètres dans l'URL). Et l'usager est informé de ses recours possibles.

## Données
- OpenData Paris (encadrement des loyers).
- API Adresse (data.gouv.fr) pour l’autocomplétion
- APUR pour aider à détecter l’époque de construction.
